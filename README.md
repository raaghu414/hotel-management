# Ranking Star Resorts - Hotel Management System

A comprehensive, full-stack hotel management solution featuring a premium guest-facing portal and a full-featured staff/admin dashboard.

## 🌟 Key Features

### Guest Portal
- **Modern UI**: High-end dark-themed aesthetic with glassmorphism and smooth transitions.
- **Room Booking**: Real-time room availability and reservation system.
- **Digital Menu**: Interactive restaurant menu with categorical breakdown and food ordering.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile devices.

### Admin Dashboard (Staff Only)
- **Role-Based Login**: Secure access for Administrators and Staff.
- **Analytics Overview**: Dynamic stats for Revenue, Occupancy, and Total Bookings.
- **Room Management**: Monitor and edit room inventory.
- **Reservation Tracking**: Real-time synchronization of guest bookings.

## 🛠️ Technical Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript.
- **Backend**: Node.js (Built-in `http` and `fs` modules).
- **Database**: JSON-based persistent storage (`data.json`).

## 🚀 Getting Started

1. **Start the Server**:
   ```bash
   node server.js
   ```
2. **Access the Sites**:
   - Guest Portal: `http://localhost:3000/`
   - Admin Dashboard: `http://localhost:3000/admin.html`
